		<a href="https://api.stackexchange.com/docs">StackOverflow API</a> using React.
	</p>

## Prerequisites

- [Node.js 20.11.1](https://nodejs.org/en)

## Getting started

```bash
git clone https://github.com/bylski9/mediporta.git
cd mediporta
npm ci
npm start
```
